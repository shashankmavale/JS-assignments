Types of Datatypes in Javascript

  • String
  • Number
  • Boolean
  • Object
  • Array
  • Function
  • Undefined
  • Null
Note:Please read the code to see examples.
  1. String
    2. Used to represent the textual data or combnations of words and numbers.
  2. Number
    3. Used to represent positive or negative numbers with or without decimal place.
  3. Boolean
    4. Boolean data types can hold only twovalues : 'True' or 'False'
  4. Object
    5. Used to store collections of data types.
  5. Array
    6. used for storing multiple values in single variable.//it is type of object.
  6. Function
    7. It is a callable object that executes a block of code.
  7. Undefined
    8. The undefined data type can only have one value-the special value undefined. If a variable has been declared, but has not been assigned a value, has the value undefined.
  8. Null
    9. A null value means that there is no value. It is not equivalent to an empty string ("") or 0, it is simply nothing.
