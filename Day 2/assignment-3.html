Explain with examples the remaining methods of String and Array

String methods

  1. String length

    2. returns the length of a string
  2. Finding a String in a String

    3. The indexOf() method returns the index of (the position of) the first occurrence of a specified text in a string
    The lastIndexOf() method returns the index of the last occurrence of a specified text in a string
  3. Searching for a String in a String

    4. The search() method searches a string for a specified value and returns the position of the match /*The difference between indexOf() and search() are - the search() method cannot take a second start posuition argument. and the indexOf() method cannot take powerful search values.*/
  4. slice() Method

    5. slice() extracts a part of a string and returns the extracted part in a new string.
  5. substring() Method

    6. substring() is similar to slice(). The difference is that substring() cannot accept negative indexes.
  6. substr() Method

    7. substr() is similar to slice(). The difference is that the second parameter specifies the length of the extracted part.
  7. replace() method

    8. replaces a specified value with another value in a string
  8. Upper and Lower Case

    9. A string is converted to upper case with toUpperCase() A string is converted to lower case with toLowerCase()
  9. concat() Method

    10. joins two or more strings
  10. trim() method

    11. removes whitespace from both sides of a string
  11. charAt() Method

    12. returns the character at a specified index (position) in a string
  12. charCodeAt() Method
    13. returns the unicode of the character at a specified index in a string
  13. split() method

    14. A string can be converted to an array with the split() method

Array Methods

  1. toString() method

    2. converts an array to a string
  2. pop() method

    3. removes the last element from an array
  3. push() method

    4. adds a new element to an array (at the end)
  4. shift()

    5. removes the first array element and "shifts" all other elements to a lower index.
  5. unshift() method

    6. adds a new element to an array (at the beginning), and "unshifts" older elements
  6. Changing Elements

    7. Array elements are accessed using their index number
  7. delete operator

    8. Since JavaScript arrays are objects, elements can be deleted by using the JavaScript operator delete
  8. splice() method

    9. The splice() method can be used to add new items to an array or to remove elements
  9. concat() method

    10. creates a new array by merging (concatenating) existing arrays
  10. slice() method

    11. The slice() method slices out a piece of an array into a new array.
